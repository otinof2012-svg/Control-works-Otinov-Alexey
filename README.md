# 🌍 Эко-справочник

**Простая консольная программа на Python, которая помогает правильно сортировать отходы и заботиться об экологии.**

Никаких сторонних библиотек — только чистый Python. Работает офлайн, данные сохраняются в текстовый файл.

---

## 📌 Возможности

- 🔍 **Поиск предмета** — введите название, получите инструкцию по утилизации
- 📋 **Все предметы** — просмотр всей базы данных
- ➕ **Добавление** — пополняйте базу своими предметами
- 🗑️ **Удаление** — удаляйте ненужные записи
- 📖 **Справка** — встроенная инструкция
- 💾 **Автосохранение** — данные хранятся в файле `eco_data.txt`

---
